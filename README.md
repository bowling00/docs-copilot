## 如何本地开发
### client
配置好`.env` 参考 `.env.example`

```
pnpm install
npm run dev
```
### server

1. 安装 docker 后，执行 `docker-compose up -d`

2. 配置好`.env` 参考 `.env.example`

```
pnpm install 
npm run start:dev
```
